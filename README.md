# Salamander Finder 

## 1. Install packages

Run "npm i" in terminal to install all packages.

```bash
npm i
```

## 1. Run Program

Run "npm run dev" in terminal to start up project.

```bash
npm run dev
```

## 1. Open webpage

Open project webpage at [http://localhost:3000](http://localhost:3000)

## 1. Start Processing

Use the settings to process videos
