# Salamander Finder 

## 1. Install packages

Run "npm i" in terminal to install all packages.

```bash
npm i
```

## 1. Run Program

Run "npm run dev" in terminal to start up project.

```bash
npm run dev
```

## 1. Open webpage

Open project webpage at [http://localhost:3001](http://localhost:3001) or the next availible port.

## 1. Select Video

Use the dropdown to select a video to process.

## 1. Change Settings

Change the color and the threshold input while using the bininzed preview image to adjust settings to your liking.

## 1. Process video

Click the button labeled "Process Video with These Settings" to start processing the video and wait for the processor to finish.

## 1. Download CSV

Once the processor is done, click the "download" button to download the csv.

## 1. Previous Jobs

If you want to download the CSV for a previous Job, click on the "Results" link to find them listed out.
